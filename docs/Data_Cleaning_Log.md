#Data Cleaning Summary 

**Dataset:** SuperMarket Sales Data  
**Tool Used:** Microsoft Excel Web  
**Objective:** Prepare clean, analysis-ready data for visualization and KPIs.

### Steps Completed
- Standardized column names and formats (Dates, Times, Numbers)
- Removed empty rows and columns
- Handled missing values appropriately
- Removed duplicate records
- Checked and handled outliers in Total, Quantity, and Unit Price
- Sorted data by Date for easier trend analysis

**Output File:** `SuperMarket Analysis CLEANED.xlsx` (stored in `/data/` folder)
